Data Types Practice

In this mini project you will practice using different kinds of data in JavaScript: strings, numbers, booleans, undefined and null.

  1. Open DevTools (Right click → Inspect → Console).
  2. Open script.js and follow each STEP comment.
  3. Write the JavaScript code under each instruction.
  4. Use console.log() and typeof to see each data type.

✨ Focus: what kind of data you are storing, not the DOM.
